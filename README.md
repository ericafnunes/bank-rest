# Bankapp

Projeto usando Quarkus e banco de dados Mysql.

RESTful API de um banco que permite:
Cadastrar Usuário
Detalhar Usuário
Listar Todos Usuários
Detalhar Contas
Cadastrar Contas
Sacar Valor da Conta
Depositar Valor da Conta
